## Adding a New Feature Page

1. Add a nav item in `src/app/dashboard/layout.tsx` (navItems array)
2. Add a slug guard in `src/app/dashboard/[feature]/page.tsx`
3. Add mock data in `src/lib/data.ts` and types in `src/lib/types.ts`
4. Track usage: `import { track } from '@/lib/analytics'; track('feature_used', { feature: 'your-slug' })`
